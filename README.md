# M365 Defender License Advisor

A lightweight, vanilla JavaScript tool designed for **CISOs** and **IT Pros** to navigate the complex world of Microsoft 365 licensing based on security requirements.

## 🚀 Key Features
- **Defender Focus:** Choose protection by vector (Endpoint, Identity, Office 365, Cloud Apps).
- **Auto-Scale:** Automatically handles the 300-seat limit between Business and Enterprise tiers.
- **Cost Estimation:** Real-time monthly budget calculation.
- **Pure Web:** No frameworks, no trackers, just HTML/CSS/JS.

## 📂 Project Structure
- `index.html`: Main UI.
- `js/data.js`: The "Source of Truth" for plan features and pricing.
- `js/engine.js`: Filtering logic and UI updates.
- `css/style.css`: Modern, clean security-focused design.

## 🛠 Usage
Simply open `index.html` in any browser or host it via **GitHub Pages**.
